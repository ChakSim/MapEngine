# MapEngine - Кастомный движок для настольных игр

**Лучше своя бензопила, чем чужой швейцарский нож!**

## О проекте
Я создал этот движок чтобы оцифровать нашу семейную настольную игру. 
Система парсит текстовые файлы и превращает их в работающую игру с анимациями.

## Архитектура
[Текстовая карта] → [Парсер] → [Игровая логика] → [Анимации]

**Структура проекта:**
- `Core/` - ядро системы
- `Parser/` - парсеры карт
- `Display/` - рендеринг и анимации
- `Services/` - сервисы и утилиты
- `Utils/` - вспомогательные функции

## Особенности
- **Читает текстовые карты** - просто редактировать
- **Умный поиск ячеек** - быстро находит что нужно  
- **Анимации движения** - фишки двигаются плавно
- **Чистая архитектура** - 64 файла с четким разделением ответственности

## Как запустить?
1. Скачайте проект
2. Откройте в Visual Studio
3. Нажмите F5

## Технологии
- C# / .NET
- Консольное приложение  
- Модульная архитектура
- ECS-подход

## Почему я это сделал?
Хотел понять как работают игровые движки изнутри. 
Начал с простого кода, потом переделал в красивую архитектуру.
